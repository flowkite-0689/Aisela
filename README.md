# 《追逐光的》故事设定集

## 📚 故事概览

《追逐光的》是一个完整的奇幻世界，包含两个相互关联的主要故事：

### 🌙 《为了什么活着》
- **主角**：洛瑟恩·暗星（魔王）
- **时间跨度**：星历180-243年
- **主题**：权力、责任、理想与现实的冲突
- **基调**：史诗悲剧，深沉厚重

### 🌱 《新枝叶的故事》
- **主角**：西里斯·弗拉尔/瑟莉安·星叶
- **时间跨度**：星历227-243年+
- **主题**：成长、友谊、身份认同、复仇与宽恕
- **基调**：从温暖明亮到深沉反思，最终回归希望
- **三部曲结构**：《日常》→《勇者》→《新生》

1. **先读** `故事总览.md` - 了解整体框架
2. **再读** `世界观设定.md` - 理解世界背景

## 🌟 核心主题

- **代际传承**：父女两代人的理想传承
- **打破循环**：从仇恨到和解的转变
- **爱的力量**：爱情、亲情、友情的治愈作用
- **成长蜕变**：在困境中的成长和选择
